- ค้นหา/แก้ไข - ใบอนุญาตจำหน่ายสินค้าในที่หรือทางสาธารณะ - ใบอนุญาตเร่ขายสินค้าในที่หรือทางสาธารณะ - ใบอนุญาตจัดตั้งสถานที่จำหน่ายอาหาร - ใบอนุญาตจัดตั้งสถานที่สะสมอาหาร - หนังสือรับรองการแจ้งจัดตั้งสถานที่จำหน่ายอาหาร - หนังสือรับรองการแจ้งจัดตั้งสถานที่สะสมอาหาร - ใบอนุญาตให้ใช้สถานที่เป็นตลาดเอกชน - กิจการที่เป็นอันตรายต่อสุขภาพ - กิจการฌาปณสถาน
- คำขอต่อใบอนุญาตจำหน่ายสินค้าในที่หรือทางสาธารณะ - คำขอต่อใบอนุญาตเร่ขายสินค้าในที่หรือทางสาธารณะ - คำขอต่อใบอนุญาตจัดตั้งสถานที่จำหน่ายอาหาร - คำขอต่อใบอนุญาตจัดตั้งสถานที่สะสมอาหาร - คำขอต่อหนังสือรับรองการแจ้งจัดตั้งสถานที่จำหน่ายอาหาร - คำขอต่อหนังสือรับรองการแจ้งจัดตั้งสถานที่สะสมอาหาร - คำขอต่อใบอนุญาตให้ใช้สถานที่เป็นตลาดเอกชน - คำขอต่อกิจการที่เป็นอันตรายต่อสุขภาพ - คำขอต่อกิจการฌาปณสถาน
