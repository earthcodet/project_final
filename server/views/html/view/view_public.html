โทรศัพท์  

1. จำหน่ายสินค้าในที่หรือทางสาธารณะ สินค้าประเภท  

2. สถานที่เร่ขายสินค้าอยู่ในพื้นที่ที่มีการประกาศผ่อนผันให้จำหน่ายสินค้าได้ในท้องที่เขต  

3. ใบอนุญาตฉบับนี้ให้ใช้ได้เฉพาะผู้ได้รับใบอนุญาตและผู้ช่วยจำหน่าย

4. ค่าธรรมเนียมฉบับละ    บาทต่อปี

5. ผู้รับใบอนุญาตต้องปฏิบัติให้ถูกต้อง ครบถ้วน ตามหลักเกณฑ์วิธีการ และ เงื่อนไขที่กำหนดในข้อกำหนดของท้องถิ่น

6. หากปรากฎในภายหลังว่าการประกอบการที่ได้รับอนุญาตนี้เป็นการขัดต่อข้อกฏหมายอื่นที่่เกี่ยวข้องโดยมิอาจแก้ไขได้
