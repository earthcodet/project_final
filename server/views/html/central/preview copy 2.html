ขอยื่นคำขอต่อใบอนุญาตฯ ประเภท


1. สถานประกอบการชื่อ

2. ตั้งอยู่เลขที่  ตรอก/ซอย  ถนน  หมู่ที่  ตำบล  อำเภอ  จังหวัด  โทรศัพท์  พื่นที่ประกอบการ

3. พร้อมคำขอนี้ข้าพเจ้าได้แนบเอกสารต่างๆมาด้วย คือ

 3.1 สำเนาบัตรประจำตัว หากเป็นนิติบุลคลให้ใช้สำเนาหนังสือรับรองการจดทะเบียนเป็นนิติบุคคล

 3.2 สำเนาทะเบียนบ้านของผู้ขอรับใบอนุญาต และทะเบียนบ้านของสถานประกอบการ

 3.3 หลักฐานแสดงว่าอาคารนั้นสามารถใช้ประกอบการได้ถูกต้องตามกฎหมาย ว่าด้วยการควบคุมอาคาร

 3.4 กรณีเจ้าของไม่สามารถมาด้วยตนเองได้ ให้ผู้รับมอบหมายอำนาจนำหนังสือมอบอำนาจที่ถูกต้องตามกฎหมาย พร้อมบัตรประจำตัว

    ประชาชนของผู้รับมอบมาแสดง

ข้าพเจ้าขอรับรองว่าข้อความในแบบคำขอนี้เป็นความจริงทุกประการ และจะปฏิบัติตามเทศบัญญัติ/ข้อบัคับและเงี่อนไงที่ระบุไว้ทุกประการ
